Getting Around:
pwd  = print working directory
ls = list contents in current working directory
cd 'name of directory' = change directory
git = lists out more git 
git status = will tell you what changes you made since your last push/pull

Setting up an SSH Key:

Syncing your computer to Github:
git pull = syncs your computer to Github

Saving your changes to Github:
3 Step Process:
  1. git add -A = this will add all changes you made to an index (package)
  2. git commit -m "your message" = Will send/commit your index to github. If you accidently forgot '-m'...
  3. git push = finalizes and pushes everything to the cloud
